# Traineeship Management App 

## About this project

This was a university project for a Software Engineering class, with the objective being the creation of an application that allows a traineeship committee, the supervising professors, and students participating in traineeships to manage what they need. Through completing the project as a team, we came to be properly acquainted with a Scrum-based approach to project development, user story implementation, Sprint reports, implementing the MVC (Model View Controller) pattern and the GoF strategy pattern (parametrized factory creates strategy).

We used the Spring framework for the purposes of developing this web-based app, MySQL for a base database management system, and JUnit and Mockito for writing automated tests.
The IDE used was Eclipse.
